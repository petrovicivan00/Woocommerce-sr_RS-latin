# WooCommerce Serbian Latin (sr_RS – Latinica)

## Compatibility

- WooCommerce: 10.5.5 and lower

## About

This package provides a fully converted Serbian Latin translation of WooCommerce.  
The original sr_RS translation has been adapted from Cyrillic to Latin script and manually reviewed for consistency and eCommerce terminology.

Designed specifically for Serbian market WooCommerce stores.

## Features

- Full Latin script conversion
- Terminology optimized for Serbian eCommerce
- Clean checkout wording
- Production-ready
- No core modifications required

## Installation

1. Upload the `.po` and `.mo` files to:

   wp-content/languages/plugins/

2. Make sure your WordPress language is set to:

   Serbian (sr_RS)

3. Clear cache if necessary.

## File Naming Convention

woocommerce-sr_RS.po 

## Notes

If you are using Loco Translate or similar plugins, make sure this file has priority and is not overridden.
